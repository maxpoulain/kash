# Objectifs de Dépenses Mensuels

## Description
Permettre aux utilisateurs de définir des objectifs de dépenses mensuels par catégorie et visualiser leur progression en fonction des transactions réelles.

## Stack
- Backend: FastAPI
- Frontend: Next.js + Tailwind + shadcn
- Database: Supabase (PostgreSQL)

## Critères d'acceptation
- [ ] L'utilisateur peut voir ses objectifs de dépense avec la progression par catégorie
- [ ] L'utilisateur peut créer un objectif de dépense mensuel pour une catégorie
- [ ] Un seul objectif par catégorie par mois
- [ ] La progression calcule toutes les transactions de la catégorie sur le mois
- [ ] Fonctionne sur desktop et mobile
