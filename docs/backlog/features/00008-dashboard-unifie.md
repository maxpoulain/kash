# Dashboard unifié — budget du mois + épargne + net worth

**Source:** [Linear PER-5](https://linear.app/personal-max-and-maria/issue/PER-5/dashboard-unifie-budget-du-mois-epargne-net-worth)

**Priority:** High

## Description

Vue d'ensemble en page d'accueil qui combine les deux dimensions de Kash : budget courant et patrimoine.

## Ce qu'on veut voir

- Solde budget du mois (alloué vs dépensé)
- Total épargne actuel
- Net worth simplifié (actifs − passifs)
- Alertes si une catégorie est dans le rouge

## Critères d'acceptance

- [ ] L'écran s'affiche en moins de 1s
- [ ] Les données sont à jour en temps réel pour les deux membres du foyer
- [ ] Fonctionne sur mobile et desktop
