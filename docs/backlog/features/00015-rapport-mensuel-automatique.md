# Rapport mensuel automatique

**Source:** [Linear PER-12](https://linear.app/personal-max-and-maria/issue/PER-12/rapport-mensuel-automatique)

**Priority:** Medium

## Description

Générer automatiquement un résumé financier en fin de mois pour le foyer.

## Contenu du rapport

- Dépenses réelles vs budget alloué par catégorie
- Taux d'épargne du mois (revenus − dépenses / revenus)
- Delta net worth vs mois précédent
- Top 3 catégories où on a dépassé le budget
- Objectifs : progression ce mois-ci

## Format de diffusion

- Affiché dans l'app (écran dédié)
- Notification push ou email en fin de mois (à confirmer)

## Critères d'acceptance

- [ ] Rapport généré automatiquement le dernier jour du mois
- [ ] Accessible dans l'historique des rapports
- [ ] Contenu clair et actionnable
- [ ] Option pour désactiver/activer
