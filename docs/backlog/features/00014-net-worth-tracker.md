# Net worth tracker — actifs vs passifs avec courbe historique

**Source:** [Linear PER-11](https://linear.app/personal-max-and-maria/issue/PER-11/net-worth-tracker-actifs-vs-passifs-avec-courbe-historique)

**Priority:** High

## Description

Calculer et afficher le patrimoine net du foyer : total actifs moins total passifs, avec courbe d'évolution dans le temps.

## Fonctionnement attendu

- Actifs : épargne + comptes courants (saisis manuellement)
- Passifs : crédits, dettes (saisis manuellement)
- Net worth = actifs − passifs
- Courbe historique mensuelle
- Delta vs mois précédent affiché en gros

## Critères d'acceptance

- [ ] Le calcul est correct et se met à jour automatiquement
- [ ] Courbe historique sur les 12 derniers mois
- [ ] Delta mensuel visible (ex: +€350 vs juin)
- [ ] Saisie manuelle des actifs et passifs
