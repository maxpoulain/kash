# Suivi de l'épargne par poste (Livret A, LEP, fonds euros…)

**Source:** [Linear PER-8](https://linear.app/personal-max-and-maria/issue/PER-8/suivi-de-lepargne-par-poste-livret-a-lep-fonds-euros)

**Priority:** High

## Description

Suivre l'épargne du foyer par poste (Livret A, LEP, fonds euros, etc.) avec saisie manuelle des montants.

## Fonctionnement attendu

- Création de postes d'épargne : nom, type, montant actuel, taux si applicable
- Mise à jour manuelle du solde (mensuelle ou à la demande)
- Historique de l'évolution du solde par poste
- Total épargne consolidé visible sur le dashboard

## Critères d'acceptance

- [ ] On peut créer autant de postes que nécessaire
- [ ] Types prédéfinis : Livret A, LEP, CEL, PEL, Fonds euros, Autre
- [ ] Historique consultable (courbe d'évolution)
- [ ] Seuls les membres du foyer voient ces données
