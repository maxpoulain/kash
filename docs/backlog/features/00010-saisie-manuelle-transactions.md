# Saisie manuelle de transactions (dépenses & revenus)

**Source:** [Linear PER-7](https://linear.app/personal-max-and-maria/issue/PER-7/saisie-manuelle-de-transactions-depenses-and-revenus)

**Priority:** Urgent

## Description

Permettre l'ajout rapide de dépenses et revenus sans connexion bancaire. C'est le mode de saisie principal du MVP.

## Fonctionnement attendu

- Formulaire rapide : montant, catégorie, date, note optionnelle
- Distinction dépense / revenu
- Possibilité de modifier ou supprimer une transaction
- Historique des transactions par mois
- Transaction partagée visible par les deux membres du foyer

## Critères d'acceptance

- [ ] Saisie en moins de 10 secondes (flow optimisé)
- [ ] Les transactions apparaissent immédiatement pour les deux membres
- [ ] Filtre par mois/catégorie dans l'historique
- [ ] Saisie sur mobile fluide (PWA)
