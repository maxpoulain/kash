# Objectifs d'épargne avec suivi de progression

**Source:** [Linear PER-9](https://linear.app/personal-max-and-maria/issue/PER-9/objectifs-depargne-avec-suivi-de-progression)

**Priority:** High

## Description

Permettre de définir une cible financière par catégorie d'épargne et suivre la progression vers cet objectif.

## Fonctionnement attendu

- Définir un objectif : nom, montant cible, date cible (optionnelle)
- Lier un objectif à un poste d'épargne ou à une catégorie budget
- Barre de progression visuelle
- Calcul automatique du montant mensuel à épargner pour atteindre la cible

## Exemples d'objectifs

- Vacances à Marseille : 1500€ d'ici juillet
- Fond d'urgence : 10000€ (sans date)
- Nouveau PC : 2000€ d'ici décembre

## Critères d'acceptance

- [ ] Création/modification/suppression d'objectifs
- [ ] Progression mise à jour automatiquement
- [ ] Calcul de l'épargne mensuelle nécessaire
- [ ] Notification quand objectif atteint
