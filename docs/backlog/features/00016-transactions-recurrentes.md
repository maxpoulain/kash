# Transactions récurrentes (loyer, abonnements, salaire)

**Source:** [Linear PER-13](https://linear.app/personal-max-and-maria/issue/PER-13/transactions-recurrentes-loyer-abonnements-salaire)

**Priority:** Medium

## Description

Pré-enregistrer les transactions qui se répètent chaque mois pour éviter la saisie manuelle répétitive.

## Fonctionnement attendu

- Créer une transaction récurrente : montant, catégorie, fréquence (mensuel, hebdo…), date de prélèvement
- Exemples : loyer le 5, Spotify le 15, salaire le 28
- Rappel ou création automatique à la date prévue
- Possibilité de modifier ou sauter une occurrence

## Critères d'acceptance

- [ ] On peut créer des transactions récurrentes
- [ ] Elles apparaissent automatiquement dans le budget
- [ ] Rappel/notification à la date
- [ ] Possible de modifier une occurrence sans impacter les suivantes
