# Commit Conventions

## Format

Conventional commits with emoji at the end:

```
<type>: <description> <emoji>
```

## Types

- `feat`: ✨
- `fix`: 🐛
- `docs`: 📝
- `style`: 💄
- `refactor`: ♻️
- `perf`: ⚡
- `test`: ✅
- `build`: 📦
- `ci`: 🔧
- `chore`: 🧹

## Rule

Never add `Co-Authored-By:` to commit messages.
